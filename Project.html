CSP Scratch Game

August 30, 2018 - September 8, 2018 in a Computer Science Principles Course

The objective of this project was to demonstrate the knowledge we gained from our Scratch unit into a game or story. My team decided to combine both a game and a story to create a story based game that follows a plot. Our story game follows the same mechanics of Space Invaders but with enemies of different skill levels, obstacles, and shields. We used this project to cement our knowledge of basic programming concepts like variable roles and loops. This game is very fun and has very few bugs that cannot be fixed based on the limitations of Scratch. My task was to code the second level and all of the last level. The documentation of our project was done by my partner and me.

CSP MIT App Inventor App

September 18, 2018 - October 6, 2018 in a Computer Science Principles Course

The objective of this project was to demonstrate our MIT App Invetor skills while at the same time strenghtening our coding fundamentals.
